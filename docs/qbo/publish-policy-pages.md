# Publish Policy Pages (for Intuit form)

Intuit requires HTTPS URLs for:

- End-user license agreement URL
- Privacy policy URL

## Quick option: GitHub Pages

1. Push this repo to GitHub.
2. In repository settings, enable Pages from the default branch.
3. Use paths:
   - `https://<your-user>.github.io/<repo>/docs/qbo/end-user-license-agreement.html`
   - `https://<your-user>.github.io/<repo>/docs/qbo/privacy-policy.html`

## Alternative

Host these files on any HTTPS domain you control and paste those URLs into Intuit.

## Before submitting

- Replace placeholder email addresses in both documents.
- Confirm owner/operator name is correct.
- Re-check "Last updated" dates.
