# Privacy Policy

Last updated: 2026-03-04

This Privacy Policy describes how the Groom Enterprises QuickBooks automation tool ("Application") collects, uses, and protects information.

## 1) Scope

This Application is a private, internal command-line tool used by Groom Enterprises to automate QuickBooks Online rule generation and validation.

It is not a public consumer service.

## 2) Information We Access

When authorized by a QuickBooks Online administrator, the Application may access:

- QuickBooks company metadata (such as company identifier / realm ID)
- Chart of accounts, classes, and account names
- Banking rule export/import content
- OAuth tokens required for API access

## 3) How Information Is Used

Information is used only to:

- Generate and validate QuickBooks bank rules
- Verify account/class/category naming consistency
- Support accounting workflow automation for internal use

We do not sell data, and we do not use data for advertising.

## 4) Data Sharing

We do not share QuickBooks data with third parties except:

- Intuit/QuickBooks API services as required for authorized API operations
- Service providers required to host internal documentation or infrastructure, if applicable
- Legal or regulatory requests when required by law

## 5) Data Storage and Retention

- OAuth tokens and configuration values are stored by the Application owner in local environment configuration.
- Generated outputs are stored in local project files unless the owner chooses to store them elsewhere.
- Data is retained only as needed for accounting operations and internal recordkeeping.

## 6) Security

We use reasonable administrative and technical safeguards, including:

- Restricted access to local systems and repositories
- Secret/token handling through environment variables
- Least-privilege API scope usage where possible

No method of transmission or storage is guaranteed 100% secure.

## 7) Your Choices

You may revoke the Application's QuickBooks access at any time from QuickBooks connected apps settings.

## 8) Children's Privacy

This Application is not directed to children and is not intended for public consumer use.

This policy may be updated periodically. The "Last updated" date indicates the latest revision.
